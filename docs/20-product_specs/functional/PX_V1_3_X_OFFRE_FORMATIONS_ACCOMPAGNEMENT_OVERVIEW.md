# Offre PixelProwlers — Formations & Accompagnement (Overview) — V1.3.x feature

## 0) North Star
PixelProwlers ne vend pas “du contenu”. PixelProwlers vend :
- **lucidité sans concession**
- **protocoles opérationnels**
- **livrables réels**
- **gates anti-régression** (orga + sécu)
- **souveraineté** (offline, privacy-first, no tracking)

## 1) Segmentation canonique (le discriminant)
Le discriminant n’est pas “avant/après crash”, mais :
- **PRÊTS** : prêts à sortir du confort illusoire, à remettre en question les croyances, à décider et exécuter.
- **PAS PRÊTS** : veulent une surcouche “as usual”.

Règle de communication :
- “Not ready” = **refus protecteur**, pas un jugement.
- Sortie propre : **Dojo gratuit + ressources + “quand revenir”**.

## 2) Architecture d’offre (V1)
### 2.1 Gratuit — DOJO (anonyme / souverain)
- accès libre sans compte
- pas de tracking / pas de pixels / pas de recaptcha tiers
- ressources téléchargeables offline (Markdown/PDF)

### 2.2 Payant — OPÉRATION 125%
- **Bootcamp présentiel 7h**
  - public : **date + ville + durée**
  - adresse exacte : **après validation Gate + confirmation**
- **Sprint 14 jours** : implémentation + gates + review “bouclier”
- livrables non négociables : voir `PX_V1_3_X_OPERATION_125_PRODUCT_SPEC`

### 2.3 Accompagnement (premium)
- adaptation au contexte réel
- implémentation assistée + hardening + audit + transfert
- maintien (rituels + revues)

## 3) Doctrine non négociable
- privacy-first, métriques agrégées uniquement
- pas de profilage, pas de CRM caché
- compartimentation : Gate ≠ facturation ≠ notes d’accompagnement
- emails neutres (pas de citations du formulaire, pas de fuites)
- anti-abus Gate sans tiers : rate limit + honeypot + validations strictes + erreurs neutres

## 4) Pages & intégrations (V1)
- Dojo (gratuit)
- Opération 125% (Gate + date/ville)
- Accompagnement
- option future : deep links /ressources via SafeDeepLinkKit (P0 déjà posé)

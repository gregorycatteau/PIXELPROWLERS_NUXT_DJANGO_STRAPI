# Kit — Dépendance mortelle

## À quoi ça sert
Identifier le point de dépendance qui peut casser la structure et préparer un vrai plan B.

## Quand l’utiliser
- Une personne, un financeur ou un outil est “indispensable”.
- Les décisions sont dictées par la peur de perdre ce point clé.
- Aucun plan B activable rapidement.

## Checklist
- [ ] Lister 3 dépendances critiques (personne, financeur, outil, presta).
- [ ] Évaluer l’impact si chaque dépendance saute 4 semaines.
- [ ] Proposer 1 plan B réaliste par dépendance (qui décide le switch ?).
- [ ] Vérifier les accès/contrats associés et les sécuriser.
- [ ] Programmer un test de switch contrôlé sur 1 dépendance.
